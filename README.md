Este proyecto desarrolla y evalúa un sistema de asignación dinámica aplicado a una inversión en el S&P 500 que mantiene un 70% de exposición estructural al mercado y gestiona de forma táctica el 30% restante en función de indicadores macroeconómicos y de mercado. El sistema no realiza trading ni predicciones, sino que genera señales analíticas de exposición orientadas a apoyar decisiones humanas, con el objetivo de mejorar el perfil riesgo-retorno a largo plazo mediante la reducción de volatilidad y drawdowns, comparando sus resultados frente a una estrategia pasiva pura desde un enfoque académico, transparente y centrado en la gestión del riesgo.
